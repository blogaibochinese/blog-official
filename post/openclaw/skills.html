OpenClaw Skills 配置指南

1. 环境准备与 Skill 发现

在开始使用 Skills 之前，请确保 OpenClaw 已正确安装并运行。

1.1 检查服务状态

openclaw gateway status

确保状态显示为"运行中"。

1.2 访问 WebUI

在浏览器中打开：http://127.0.0.1:9876 或 http://localhost:9876

2. 常用 Skills 安装

2.1 搜索类 Skills

clawhub search <关键词>

2.2 安装 Skill

clawhub install <skill-name>

2.3 热门 Skills 示例

  • brave-search：Brave 浏览器搜索
    • clawhub install brave-search
  • filesystem：文件系统操作
    • clawhub install filesystem
  • browser：网页浏览
    • clawhub install browser
  • terminal：终端命令执行
    • clawhub install terminal

3. 列出已安装 Skills

clawhub list

4. 更新 Skills

clawhub update <skill-name>

5. 删除 Skills

clawhub remove <skill-name>

6. Skill 配置

安装后，某些 Skills 需要额外配置。

6.1 配置文件位置

nano ~/.openclaw/skills/<skill-name>/config.json

6.2 配置示例：Brave Search

{
  "api_key": "your-brave-api-key"
}

7. 使用 Skills

在对话中直接自然语言描述任务，OpenClaw 会自动调用相应的 Skill。

例如：

  • "帮我搜索最新的科技新闻"
  • "读取当前目录的文件列表"
  • "打开 https://example.com 并总结内容"

8. Skill 开发入门

创建自定义 Skill：

mkdir -p ~/.openclaw/skills/my-skill
cd ~/.openclaw/skills/my-skill

8.1 Skill 结构

skill.json
index.js
README.md

8.2 skill.json

{
  "name": "my-skill",
  "description": "My custom skill",
  "version": "1.0.0",
  "entry": "index.js"
}

9. 常见问题

  • Skill 不工作？检查日志：tail -f ~/.openclaw/logs/openclaw.log
  • 权限问题？确保有正确的文件权限
  • 网络问题？检查网络连接
