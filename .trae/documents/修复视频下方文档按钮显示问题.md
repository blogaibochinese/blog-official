## 问题分析
当前视频播放页面布局存在以下问题：
1. `.container` 设置了固定高度 `80vh` 和 `overflow: hidden`
2. `.video-player` 使用 `flex: 1` 占据了所有可用空间
3. `.video-player-container` 作为 flex 容器，视频播放器占据过多空间，导致按钮被挤出或隐藏

## 解决方案
调整 CSS 样式，确保按钮有足够空间显示：
1. 修改 `.video-player-container` 的 flex 布局，确保视频播放器和按钮都能正常显示
2. 调整视频播放器的高度计算方式，给按钮留出空间
3. 确保 flex 布局正确计算子元素高度

## 具体修改
1. 给 `.video-player` 添加 `max-height` 限制，使用 `calc()` 函数计算可用高度
2. 调整 `.video-player-container` 的布局，确保按钮不会被挤压
3. 优化视频播放器的高度自适应逻辑
