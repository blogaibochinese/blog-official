### 问题分析
右侧悬浮的微信客服和搜索弹窗点击后显示在页面底部，而不是作为弹窗显示。原因是这些弹窗使用了自定义的类名（`search-modal`和`wechat-modal`），但CSS中没有为这些类定义模态框样式。

### 解决方案
1. 为`search-modal`和`wechat-modal`类添加与现有`.modal`类相同的模态框样式
2. 为`search-popup`和`wechat-popup`类添加与现有`.modal-content`类相同的弹窗内容样式
3. 确保弹窗内容的z-index值足够高，以正确显示在页面中央

### 修复步骤
1. 在CSS样式中添加以下代码：
   ```css
   /* 搜索和微信弹窗样式 */
   .search-modal,
   .wechat-modal {
       display: none;
       position: fixed;
       top: 0;
       left: 0;
       width: 100%;
       height: 100%;
       background-color: rgba(0, 0, 0, 0.6);
       z-index: 2000;
       justify-content: center;
       align-items: center;
       backdrop-filter: blur(5px);
   }

   .search-popup,
   .wechat-popup {
       background: white;
       border-radius: 20px;
       padding: 35px;
       max-width: 500px;
       width: 90%;
       box-shadow: 0 10px 40px rgba(0, 0, 0, 0.3);
       animation: slideDown 0.3s ease;
       text-align: center;
   }

   .wechat-qr {
       max-width: 200px;
       height: auto;
       margin: 20px auto;
       border-radius: 8px;
   }
   ```

2. 确保弹窗内的关闭按钮和内容布局正确

### 预期效果
点击右侧悬浮的微信客服或搜索图标时，会在页面中央显示对应的弹窗，背景半透明模糊，弹窗内容居中对齐，并且可以通过点击关闭按钮或背景关闭弹窗。
