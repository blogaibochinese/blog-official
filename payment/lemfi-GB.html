国内用户注册教程

步骤（5-10分钟）：

  • 1. 下载LemFi App（iOS/Android）。
  • 2. 使用英国手机号注册（推荐Kitesim教程，APP内直接查收验证码,操作简易）。
  • 3. 注册时必须用英国IP（免费试用【专线：https://dedicatedlines.top/】，问客服要英国IP）。
  • 4. 选择“英国”地区开户（只能选一个地区）。
  • 5. 填写邮箱 + 身份认证（护照/海外驾照/居留证任意一项）。
  • 6. 输入专属推荐码（HUAN7B27）获取首笔福利和兑换奖金
  • 7. 注册完成后，可切换国内IP正常使用。

LemFi入金教程

LemFi提供个人专属 ClearBank 同名账户（含Sort Code + Account Number / IBAN）。

常用方式：

  • 1. 在LemFi App → “Add Money” → 查看ClearBank账户信息。
  • 2. 从你的英国同名账户转账：
    • iFAST（教程）：登录iFAST App → Transfer → 输入LemFi ClearBank账户 → 确认同名 → 即时到账。
    • Krak英区（教程）：先提GBP到iFAST，再转LemFi。
  • 3. 资金到LemFi后立即可用。

LemFi出金教程

  • 1. App内点击“Send Money” → 选择人民币 → 输入金额。
  • 2. 选择收款方式：
    • 银联卡（首推，风控最低，10分钟到）。
    • 微信（3-5分钟，需开通微汇款）。
    • 支付宝（3-5分钟，需开通闪速收款）。
  • 3. 支持汇给自己/父母/朋友。
  • 4. 确认实时汇率 → 支付 → 3-10分钟到账。

年度额度：银联卡额外5万美金 + 支付宝/微信共享80万人民币。
